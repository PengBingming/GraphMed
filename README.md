# GraphMed
A Shiny Web Application to Analyze and Visualize Medical Data

This application is free at website: https://shiny.chcmu.com.cn/graphmed/

Mutil-language file is placed in the tool.path/lang/info/ directory

The main panel scripts are saved on   GraphMed/graphmed/
Shiny tools are categorized into 3 components: Basic graphic, Clinic & Lab. tools and Omics data visualization. It's  panel scripts are saved on ggplot2, clinic, lab, rnaseq and enrich.

Demo data for each visualization function (files are placed in the tool.path/www/ directory)

deseq2/exp.csv and group.csv for DESeq2
heatmap/heatmap.csv for heatmap.
......
